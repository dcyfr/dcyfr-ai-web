### Major Changes

- Initial v1.0.0 release of @dcyfr/ai-web

  This marks the first stable release of the production-ready Next.js web application template.

  Features:
  - Full-stack Next.js 15 application with App Router
  - JWT authentication and authorization
  - Drizzle ORM integration with SQLite/PostgreSQL support
  - Tailwind CSS with custom design system
  - Comprehensive UI component library
  - Production-ready API routes and database patterns
  - Complete test suite with 73 passing tests
  - TypeScript-first development experience
  - Security best practices and sanitization
  - Responsive design and performance optimizations

  Breaking Changes:
  - Initial release - no prior versions to compare against

  Documentation:
  - Complete API documentation with examples
  - Component usage patterns and composition guides
  - Database operations and query examples
  - Authentication and authorization patterns
  - Deployment and configuration guides
